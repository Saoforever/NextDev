Start Here

This page provides guidance on how to begin your journey into web development.

Career Pathways

Explore different roles in web development such as frontend developer, backend developer, and full stack developer. Understanding these roles helps you decide what direction you want to take in your learning journey.

Setup Guide

Learn how to set up your development environment including installing Visual Studio Code, Git, and a modern browser. Having the right tools installed will make learning and building projects much easier.

Foundational Concepts

Before building full websites, it is important to understand the core concepts of web development including HTML structure, CSS styling, and how browsers render webpages.
